# SmartEduConnect

SmartEduConnect is a full-stack academic management portal that enables students and faculty to manage courses, attendance, assignments, and academic performance through a centralized system.

## Tech Stack

Frontend:
- React.js
- HTML
- CSS

Backend:
- Java Servlets
- JSP
- Apache Tomcat

Database:
- MySQL
- JDBC

## Features

- Student Dashboard
- Faculty Dashboard
- Attendance Tracking
- Assignment Management
- Course Management
- Role Based Access Control
